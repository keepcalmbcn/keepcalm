# Luma Event Description - Keep Calm Running

---

**English Speaking Running - All Levels**

A relaxed group run along one of Barcelona's best routes, from flat seafront paths to hillside trails. Faster runners circle back and check in, so nobody gets left behind.

**The essentials**

- 📅 Runs announced in the WhatsApp group
- 📍 Meeting point and route shared in advance
- 💶 Free
- 👥 All paces welcome, 18+

**How to join**

RSVP here, then join our running WhatsApp group at keepcalm.fit/luma/running - meeting points, routes, and distances are shared there.

Keep Calm is a not-for-profit, volunteer-run community for English speakers in Barcelona.
