# Organiser Guide

This is the playbook for running a sport or activity in Keep Calm. Whether you're taking over an existing sport or starting something new, this guide covers what you need to know.

---

## What It Means to Be an Organiser

### The Role

As an organiser, you're responsible for making your sport happen reliably. You're not doing it alone—you have other organisers, bookers, and the community behind you—but you're the one who makes sure the weekly rhythm doesn't break.

### The Commitment

- **Time**: 1-3 hours per week depending on sport complexity
- **Consistency**: Your sport needs you (or a backup) every week
- **Communication**: Responsive to WhatsApp, especially around event days
- **Duration**: Aim for at least 3-6 months to give the role continuity

### What You Get

- Recognition in the community
- Input on how your sport runs
- The satisfaction of making something happen for others
- A pathway to broader community leadership if you want it

### What You Don't Get

- Commercial promotion opportunities (see [Partnerships-Reciprocity.md](partnerships-reciprocity.md))
- Payment (we're non-commercial)
- Exemption from community rules

## The Weekly Cycle

Every sport follows a similar rhythm. Adapt the timing to your activity.

### Generic Weekly Template

| Day | Task |
|-----|------|
| **Day 1** | Post poll/announcement for upcoming session |
| **Day 2-4** | Monitor responses, answer questions |
| **Day 5** | Close poll, organise games/groups/teams |
| **Day 6** | Post final details, confirm with bookers/helpers |
| **Day 7** | Event day - be available, handle issues |
| **Day 7+** | Follow up on no-shows, thank helpers, note improvements |

### Example: Padel (Sunday Sessions)

| Day | Task |
|-----|------|
| **Saturday** | Post poll for NEXT Sunday (1 week ahead) |
| **Sun-Tue** | Monitor poll responses, send reminders |
| **Tuesday** | Close poll, build games based on times/levels |
| **Wed-Thu** | Post games, assign bookers |
| **Fri-Sat** | Confirm courts booked, check weather |
| **Sunday** | Games happen, be available for issues |
| **Monday** | Follow up on any problems, update records |

### Example: Ping Pong (Thursday Sessions)

| Day | Task |
|-----|------|
| **Tuesday** | Post poll for Thursday |
| **Wednesday** | Close poll, confirm numbers |
| **Thursday** | Session happens, be there or have backup |

## Core Responsibilities

### 1. Posting Polls/Announcements

**What to include:**
- Date and time of the session
- Location (with Google Maps link if possible)
- Cost (if any)
- What to bring
- Deadline to respond
- Contact for questions

**Poll template:**
```
🎾 [SPORT] - [DAY] [DATE]

📍 Location: [Venue name]
🕐 Time: [Time range]
💰 Cost: [Price per person or Free]
📅 Poll closes: [Day/time]

Vote below for your availability!
```

**Tips:**
- Post at consistent times each week
- Pin important messages
- Use emojis sparingly but consistently (helps scanning)
- Tag @everyone sparingly—reserve for important announcements

### 2. Organising Games/Groups

For sports that need team formation (Padel, Tennis, Volleyball):

**Considerations:**
- Skill level matching where relevant
- Time slot preferences
- Gender balance if applicable
- Mixing regulars with newcomers
- Ensuring everyone who voted gets a spot

**Process:**
1. Export or note poll results
2. Group players by time preference
3. Balance by skill level within groups
4. Assign booker role to reliable players
5. Post assignments clearly

### 3. Managing Bookers

If your sport uses court bookings:

**Booker responsibilities:**
- Book the court promptly after assignment
- Share booking link with all players
- Monitor that all players join
- Handle any issues with their court
- Report problems to organiser

**What to tell bookers:**
- Book within [X hours] of assignment
- Use "Share Externally" for Playtomic links
- Tag all players when sharing
- Let organiser know if anyone hasn't joined/paid

### 4. Handling the Day

**Before the session:**
- Check weather (for outdoor sports)
- Post any updates or reminders
- Confirm bookers have booked
- Be available on WhatsApp

**During the session:**
- Be present if possible (or have a session host)
- Welcome newcomers
- Handle any issues that arise
- Take a photo if possible (for social media)

**After the session:**
- Thank participants
- Note any issues to address
- Follow up on no-shows
- Share photos with social media coordinator

### 5. Handling Problems

**No-shows:**
1. Note who didn't show
2. Check if they gave notice (check chat history)
3. If no notice: private message asking what happened
4. If pattern emerges: apply strike system (see [Event-Rules.md](event-rules.md))

**Last-minute cancellations:**
1. Post in group asking for replacement
2. Check reserves/waiting list
3. If no replacement: redistribute players or cancel court
4. Note for future reference

**Weather issues:**
1. Check venue policy on cancellations
2. Communicate clearly to group
3. Don't call venues directly—coordinate through proper channels
4. See sport-specific rain policies

**Conflicts between players:**
1. Don't take sides publicly
2. Speak to parties privately
3. Escalate to core team if serious
4. See [Community-Conduct.md](community-conduct.md)

## Communication Guidelines

### WhatsApp Best Practices

**Do:**
- Keep messages clear and scannable
- Use consistent formatting
- Pin important information
- Respond to questions promptly
- Thank people publicly

**Don't:**
- Spam the group with multiple messages
- Get into arguments in the group
- Share personal drama
- Post off-topic content
- Use excessive voice notes

### Tone

- Friendly but clear
- Inclusive and welcoming
- Positive but realistic
- Firm on rules when needed

### Handling Difficult Messages

If someone complains publicly:
1. Acknowledge their concern briefly
2. Take the conversation to DM
3. Don't get defensive in the group
4. Resolve privately, update group if needed

## Working with Other Organisers

### Your Sport Has Multiple Organisers

- Agree on who does what each week
- Have a backup plan when someone is away
- Communicate handovers clearly
- Don't step on each other's announcements
- Present a united front to the group

### Coordinating with Other Sports

- Avoid clashing announcements (stagger timing)
- Cross-promote beginner sessions
- Share learnings that might help others
- Refer players to other sports they might enjoy

### The Core Team

For issues beyond your sport:
- Community-wide policy questions
- Serious conflicts
- Cross-sport coordination
- Strategic decisions

## Onboarding New Members

When someone new joins your sport:

### First Message
Send a welcome DM:
```
Hi [Name], welcome to Keep Calm [Sport]!

Quick start:
1. Watch for the poll on [day] for upcoming sessions
2. Vote if you can make it
3. Check the games post on [day] for your assignment
4. Show up and have fun!

Your level: Let me know your experience so we can match you appropriately.

Questions? Just ask here or in the group.
```

### First Session
- Introduce them to others
- Pair them with a friendly regular
- Check in during/after to see how it went
- Follow up to encourage them to return

## Record Keeping

### What to Track

**Minimum:**
- Who played each session (for strike system)
- No-shows and reasons
- Strike counts

**Recommended:**
- Player skill levels
- Contact details beyond WhatsApp
- Equipment borrowed/returned
- Venue contacts and booking details

**Nice to have:**
- Session attendance trends
- New member conversion rates
- Feedback received

### Where to Keep It

- Shared document (Google Sheets) accessible to all sport organisers
- Backed up regularly
- Handover documentation updated

## When Things Go Wrong

### You Can't Make It This Week

1. Notify your co-organisers ASAP
2. Hand over specific tasks
3. Be available by phone for emergencies
4. Brief them on anything unusual this week

### You Need to Step Back

1. Give as much notice as possible (ideally 2-4 weeks)
2. Document everything you do
3. Help recruit and train replacement
4. Do a proper handover (see below)
5. Stay available for questions during transition

### Your Sport Is Struggling

Signs:
- Declining attendance
- Same people every week
- Difficulty filling sessions
- Complaints about the experience

Actions:
1. Talk to regulars about what's wrong
2. Survey people who stopped coming
3. Try something different (venue, time, format)
4. Ask core team for help
5. Consider pausing vs pushing through

## Handover Process

When you're leaving the role or taking extended leave:

### Handover Checklist

**Documentation:**
- [ ] Weekly process documented step-by-step
- [ ] Login credentials shared securely (if any)
- [ ] Venue contacts and booking procedures
- [ ] Supplier/equipment contacts
- [ ] Player records and strike counts
- [ ] Any ongoing issues noted

**People:**
- [ ] Replacement identified and agreed
- [ ] Introduction to venue contacts
- [ ] Introduction to regular players
- [ ] Shadowing period completed (1-2 weeks)
- [ ] Announced to the group

**Systems:**
- [ ] Access to shared documents transferred
- [ ] WhatsApp admin rights updated
- [ ] Any accounts/logins handed over
- [ ] Recurring calendar reminders shared

### Shadowing

Ideal handover includes:
- **Week 1**: New organiser observes, you run everything
- **Week 2**: New organiser runs with your backup
- **Week 3**: New organiser runs solo, you're available for questions

## Starting a New Sport

If you're launching something new for Keep Calm:

### Before You Start

1. **Check demand**: Is there interest in the group?
2. **Check competition**: Are there existing Barcelona communities doing this well?
3. **Check logistics**: Can you find venues, equipment, enough players?
4. **Check yourself**: Can you commit to running this for 3-6 months minimum?

### Proposal to Core Team

Share:
- What sport/activity
- Proposed day/time/frequency
- Venue options and costs
- Equipment needed
- Your plan for first 4 sessions
- How you'll recruit players

### Launch Checklist

- [ ] Announced in main Keep Calm group
- [ ] WhatsApp group created (or channel in community)
- [ ] First session date set
- [ ] Venue confirmed
- [ ] Equipment sourced
- [ ] Beginner-friendly first session planned
- [ ] README.md created for the sport folder
- [ ] Added to SCHEDULE.md

## Quick Reference

### Weekly Must-Dos
1. Post poll/announcement on schedule
2. Organise games/teams before deadline
3. Be available on event day
4. Follow up on issues after

### Monthly Should-Dos
1. Review attendance trends
2. Welcome check on recent new members
3. Update any stale documentation
4. Coordinate with other organisers

### Quarterly Nice-to-Dos
1. Gather feedback from regulars
2. Try something new (venue, format, social)
3. Review and improve processes
4. Recognise active helpers publicly
