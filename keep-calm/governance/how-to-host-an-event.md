# How to Host a Keep Calm Event

A step-by-step schematic for running a Keep Calm event from scratch,
without needing an admin to do it for you. If you're taking on the
[Event Organiser](role-descriptions.md#event-organiser) role for a one-off
event (social, sport taster, hike, etc.), this is your playbook. Running a
recurring weekly sport instead? See the
[Organiser Guide](organiser-guide.md).

---

## Who can host

Anyone in [Keep Calm Core or Helpers](../operations/whatsapp-groups.md) can
host an event. You don't need to be an admin of any WhatsApp group to do
this - see the pinning step below for what to do if you're not one.

## 1. Before you post anything

- Pick an activity, date, time, and location.
- Check with the admins in **Keep Calm Organisers** before posting
  anywhere else. It's the admins' job to check your proposed date/time
  against everything else already planned and flag any clash - don't rely
  on spotting a conflict yourself. This is also a good moment to see if
  anyone wants to help or co-host. Organising alone is harder than it
  needs to be.
- Decide if this needs a **capacity limit or payment** (e.g. a court
  booking, a ticketed venue). If yes, you'll need a Google Form or similar
  sign-up (see [hiking/planning.md](../../hiking/planning.md) step 6 for a
  template). If no, a plain WhatsApp Event with a "Going" RSVP is enough,
  no form needed.

## 2. Create the WhatsApp Event

Use WhatsApp's built-in **Create Event** feature (in the relevant group:
group name -> the `+`/attachment icon or group info -> Create Event).
Fill in:

- **Event name** - short and clear, e.g. "Sunday Beach Volleyball". This is
  the line that shows up in the group's pinned-message preview at the top
  of the chat, so it needs to stand alone with no other context: include
  the activity and day/date, don't rely on people opening the full post to
  understand what it is.
- **Date** and **Start time**
- **Location** - a pinned map link if it's not obvious
- **Description** - what to expect, what to bring, cost if any, and a link
  to the sign-up form if you're using one
- **Photo** - always attach one. Ideally a real photo from a previous
  edition of this activity (social proof of people actually having a good
  time beats a generic graphic or stock image every time).

See [social/annual_celebration/whatsapp_event.md](../../social/annual_celebration/whatsapp_event.md)
for a filled-in example.

**If you can't create an Event** (some groups restrict this to admins),
post the same details as a regular message instead, or ask in **Keep Calm
Organisers** for an admin to create it for you and add you as a co-host if
WhatsApp supports it for that group. If posting as a regular message, the
same rule applies: the first line is what shows in the pin preview, so
lead with the clear, standalone version of the event name, not a greeting
or a "hey everyone."

## 3. Where to post it

- Post in the **primary group for the activity** (see
  [whatsapp-groups.md](../operations/whatsapp-groups.md) for the full
  list).
- Cross-post to **Keep Calm Social** if it's open to the wider community
  rather than one activity's regulars.
- Don't post the same event in more than 2-3 groups - it reads as spam.

## 4. Managing RSVPs

- People mark themselves **Going / Maybe / Not going** on the WhatsApp
  Event. Keep an eye on numbers if capacity matters for your logistics.
- **Allow plus ones by default.** Only restrict them if you have a hard
  capacity limit (a booked court, a ticketed venue, a fixed number of
  seats) - the default should be the more the merrier.
- **If there's no money involved, expect drop-offs.** A chunk of people
  who marked "Going" simply won't show up on the day - that's normal for
  a free, casual event, not a sign anything went wrong. Don't take it
  personally or chase people down over it.
- **If something is being reserved on your behalf** (a court, a pool
  table, tickets, a table at a venue), get payment upfront, before the
  event, not on the day. Money is what actually converts a "Going" into a
  real commitment - a WhatsApp RSVP alone doesn't. This is the same rule
  already used for paid padel sessions: see
  [padel/absolute-beginners/leader.md](../../padel/absolute-beginners/leader.md)
  ("a player is only confirmed when they've paid - a yes in WhatsApp does
  not count").

## 5. Get it pinned

Once your event is posted, ask an admin (in **Keep Calm Moderators** or
**Keep Calm Organisers** - tag one directly rather than posting and hoping
someone notices) to pin it. Pinning is an admin action, and it's
temporary: the admin pins it for the run-up to the event and unpins it
once it's passed, so pinned messages don't pile up in the group over
time.

## 6. On the day

- Arrive early, be visible, welcome people as they turn up.
- Do a headcount if it matters for logistics (courts, capacity, transport).
- Take a group photo - it's the easiest way to fill step 6.
- Handle it if plans change on the day (delay, venue issue, weather) -
  post an update in the same group you used to announce it.

## 7. After the event

Send a recap message using the
[Post-Event Recap Template](../marketing/post-event-recap-template.md) in
the group(s) you posted in. It already covers:

- Thanking attendees and any co-hosts/helpers
- Asking people to share photos/videos, tag **@keepcalmbarcelona** and use
  **#keepcalmbarcelona** on Instagram so the community account can repost
- A feedback form link, if you have one
- What's coming next, if there's a follow-up planned

Post your own best photo to the group too, even before people send theirs
in - it sets the tone.

---

## Quick checklist

- [ ] Checked with admins in Organisers group for clashes / help co-hosting
- [ ] Decided if a sign-up form is needed
- [ ] Created the WhatsApp Event with name, date, time, location, description
- [ ] Event name is clear standalone (it's what shows in the pin preview)
- [ ] Attached a photo, ideally social proof from a past edition of this activity
- [ ] Posted in the right group(s), not more than 2-3
- [ ] Decided plus-one policy (allowed by default unless capacity-limited)
- [ ] Collected payment upfront if anything is being reserved (court, table, tickets)
- [ ] Asked an admin to pin it
- [ ] Took a photo on the day
- [ ] Sent the post-event recap with the Instagram call-to-action
