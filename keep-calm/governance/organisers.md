# Keep Calm Organisers

Our community is run by volunteers who organise events across different sports. Each sport has one or more organisers responsible for making it happen each week. If you have questions about a specific sport or want to help organise, reach out in the relevant WhatsApp group.

## Sport Organisers

| Sport | Responsibilities |
|-------|-----------------|
| **Padel** | Saturday poll posting, Sunday game organisation, venue coordination, player management |
| **Volleyball** | Beach session coordination, equipment management, event announcements |
| **Ping Pong** | Thursday session organisation, player coordination, equipment |
| **Tennis** | Tuesday game organisation, court bookings, player assignments |
| **Hiking** | Monthly hike planning, route selection, event coordination, safety management |
| **Social Events** | Social event planning, venue selection, seasonal events, post-game meetups |

## Want to Help?

We're always looking for more organisers and helpers! If you're interested in:
- Leading a new sport or activity
- Helping with existing sports
- Taking on admin duties
- Organising social events

Ask in the relevant WhatsApp group or speak to an existing organiser.
