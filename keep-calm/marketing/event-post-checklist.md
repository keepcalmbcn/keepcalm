# Event Post Checklist

Use this before sending any event post (WhatsApp, Instagram, Facebook, Meetup, Reddit). It is the quick "did I cover everything?" list so members do not have to ask follow-up questions.

A good event post answers, up front, every question a member would otherwise ask in the chat. If someone has to reply "where is it?" or "how much?", the post was missing something.

## The essentials (every post needs these)

- [ ] **What is it** - the activity and a one-line hook (e.g. "Sunday padel", "Thursday ping pong social").
- [ ] **When is it** - day, date, and start time. Add the end time or rough duration if known.
- [ ] **Where is it** - venue name plus a maps link or address. Note the meeting point if it is not obvious.
- [ ] **Who it is for** - level (all levels / beginners welcome / intermediate+) and any group it suits.
- [ ] **How to join** - the sign-up link, poll, or form. Be explicit: "react with 👍", "fill in the form", "comment below".
- [ ] **Cost** - the price per person, or "free". Never leave cost ambiguous.

## Often needed (add when relevant)

- [ ] **Spaces / capacity** - how many places, and whether it is first come first served.
- [ ] **Sign-up deadline** - when people need to confirm by (e.g. "by Friday 8pm").
- [ ] **What to bring** - racket, ball, water, sports kit, indoor shoes, cash, swimwear, etc.
- [ ] **What is provided** - equipment, balls, court booking, spare rackets for newcomers.
- [ ] **Skill / format** - singles vs doubles, rotation, teams, casual vs competitive.
- [ ] **Weather / rain policy** - what happens if it rains, and where the call is announced.
- [ ] **Accessibility / getting there** - nearest metro/bus, parking, or how to find the group on arrival.
- [ ] **Who to contact** - the organiser for that session and how to reach them with questions.

## Before you hit send

- [ ] **Sign-up link works** - open it and check it points to the right event and is not full/closed.
- [ ] **Right group / channel** - posting to the correct WhatsApp group or platform.
- [ ] **Formatting** - WhatsApp bold (`*text*`), clear emoji headers, scannable on a phone.
- [ ] **No private data** - no member names, phone numbers, payment details, or private links in a public post.
- [ ] **Tone** - warm, inclusive, no pressure. "Just turn up" energy.
- [ ] **Cross-post check** - if sharing outside the community, keep it neutral with no Keep Calm branding or links (see external-post guidance).

## Minimal template

```
🎾 *What:* [Activity + one-line hook]
📅 *When:* [Day, date, start time]
📍 *Where:* [Venue + maps link]
👥 *Who:* [Level / who it's for]
💶 *Cost:* [Price or "free"]
🎒 *Bring:* [What to bring]
✅ *Sign up:* [Link / poll / how to join] - [deadline if any]
❓ Any questions, just ask!
```
