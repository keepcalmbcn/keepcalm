Using the template below, arrange padel games based on the attached CSV file. Follow these rules when creating the games:

**Data cleanup:**
1. Format all player names as "Firstname L" (first name + last initial only)
2. Make sure all numbers that don't have a country code use Spain's +34

**Player Assignment Rules:**
3. Assign players to their correct time slots based on CSV data
4. Assign players to their correct ability levels based on CSV data
5. Create additional courts when more than 4 players are available for the same level/time slot
6. Within each level/time slot, review player "special requests/preferences" and fulfill where possible
7. After fulfilling special requests, prioritize creating mixed doubles courts (2 females + 2 males)
8. Fill empty slots with "*Spare Place*" (italicized)

**Booking Rules:**
9. Designate one player from each match (normally the first listed) as the "booker" unless the player stated they didn't want to book
10. **CRITICAL:** Each player can only be assigned as booker once - do not reuse the same person to book multiple courts. If a player plays in both time slots, only assign them as booker for ONE court.

**Formatting Requirements:**
11. Use clear time slot headers: `# ⏰ 16:00 - 17:30` and `# ⏰ 17:30 - 19:00`
12. Use level headers with emojis: `## 🟢 BEGINNER`, `## 🟡 IMPROVERS`, `## 🟠 INTERMEDIATE`, `## 🔴 ADVANCED`
13. Add gender indicators to courts: "Mixed 👫", "Ladies 👩", "Men's 👨" (only when all 4 players match)
14. Use separators `---` between major sections
15. Date format: "Sunday [Day] [Month] 2025" (ensure correct year)
16. Include a "## ✅ Special Requests Fulfilled" section at the end documenting how special requests were handled

**Verification Process:**
After generating the initial arrangement, systematically verify:
- Every player from the CSV appears in their correct time slot(s)
- Every player has all the games they signed up for
- No player is double-booked at the same time
- **No player is assigned as booker more than once**
- Special requests have been accommodated where possible
- Mixed doubles courts are prioritized before single-gender courts
- Booking assignments follow the rules above
- Double check the mixed matches have 2 males and 2 females not 3 females or 3 males

**Output Format:**
Please provide the output as markdown, clearly showing:
- Court assignments for each time slot and level with proper formatting
- Booker designation for each court
- Date should be replaced with next Sunday's date (correct year: 2025)
- Any numbers that don't have a country code can you assume they are +34
- The output should be in markdown but no tables as they are not WhatsApp friendly
