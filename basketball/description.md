# Keep Calm Basketball - Group Description

The following is used as the WhatsApp group description.

---

Keep Calm Basketball - part of the Keep Calm community for English speakers in Barcelona.

Pick-up basketball around Barcelona! We have a regular weekly game plus ad-hoc games throughout the week. Anyone can post a game - just say where and when and see who's in.

Regular game: Wednesdays around 19:00 at the public courts near Sants (Parc de l'Espanya Industrial). Usually 3 on 3 half court, adjusted to numbers.

Bring a ball if you have one! Pick-up games only happen if someone brings one, so check the group before heading down.

All levels welcome, from complete beginners to experienced players. It's social, not a tournament.

Cost: Free! We use public courts.

Open to adults only (18+).
