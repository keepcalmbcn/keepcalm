# Keep Calm Watersports - Group Description

The following is used as the WhatsApp group description.

---

Keep Calm Watersports - part of the Keep Calm community for English speakers in Barcelona.

Ad-hoc water-based experiences: stand-up paddleboarding to start, with more to come if there's interest. Events run whenever an organiser has arranged a venue or a good rate, so watch this space for announcements.

Spots are often limited, so sign up quickly once an event drops.

All abilities welcome, no experience needed. Open to adults only (18+).
