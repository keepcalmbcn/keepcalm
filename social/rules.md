# Keep Calm Social - Group Rules 📋

Welcome to Keep Calm! We're Barcelona's home for English speakers to stay active, make friends, and feel like they belong. To keep things running smoothly, please follow these guidelines.

> **These rules add to — but don't replace — our community-wide policies. Please read these first if you haven't already:**
> - [WhatsApp Rules](https://github.com/keepcalmbcn/keepcalm/blob/main/keep-calm/rules/whatsapp-rules.md) — group communication, unsolicited DMs, referral links, spam
> - [Community Conduct](https://github.com/keepcalmbcn/keepcalm/blob/main/keep-calm/rules/community-conduct.md) — respect, harassment, advertising, conflict resolution
> - [Event Rules](https://github.com/keepcalmbcn/keepcalm/blob/main/keep-calm/rules/event-rules.md) — commitment, cancellations, the strike system

## 1. Event Posting

✅ *"I'm going to X, who wants to join?"* Always welcome. Casual invites to do things together are what this group is about.

✅ **Official Keep Calm events** posted by admins/organisers.

❌ **External event links, flyers, Meetup/Luma links, or Instagram posts.** Not permitted without admin approval. If you'd like to share something, message an admin first.

❌ **Promoting other communities, groups, or platforms.** This includes subtle recruitment (e.g. "dm me to join"). We put a lot of volunteer effort into building this space and ask that you respect that.

## 2. Ticketed / Paid Events

If you want to suggest a paid event (festival, brunch, experience, etc.) post it as a casual suggestion:

*"Could we consider Palo Alto Market on the 8th?"*

If there's interest, an admin will create an official event with the relevant details and links. Don't post ticket links or payment details directly.

## 3. Commercial Activity

No buying/selling (whatsoever) products, tickets, services, flats, or posting affiliate/referral links. 

If you run a business relevant to the community (e.g. a local venue or activity), reach out to an admin.

## 4. Be Sound

- Be welcoming to newcomers. Remember what it felt like when you first arrived in Barcelona.
