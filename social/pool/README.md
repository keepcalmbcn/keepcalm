# Keep Calm Pool 🎱

Welcome to Keep Calm Pool! Casual pool and snooker nights around Barcelona,
part of the larger **Keep Calm** not-for-profit community for English
speakers in Barcelona, built on values of inclusivity, reliability, and
collaboration. Everyone is welcome.

**Venue and organiser vary each time.** Barcelona has pool/snooker venues
all over the city, and whoever picks up hosting a night chooses a venue
that works for them (their neighbourhood, a place they know, whatever's
convenient). There's no fixed "home" venue and no single fixed organiser -
see [How to Host a Keep Calm Event](../../keep-calm/governance/how-to-host-an-event.md)
if you want to run the next one.

## How It Works

- **Drop-in window, not a fixed start** - typically turn up any time within
  a set window
- **No booking and no cost to RSVP** - just mark yourself as going in the
  WhatsApp event so there's a headcount
- **Pay as you go** - table time and drinks are paid at the venue and split
  between players
- **Grab a table when one is free** and play as tables open up - all skill
  levels welcome

The exact venue, date, and time change each time - check
[event.md](event.md) for whatever's next.
